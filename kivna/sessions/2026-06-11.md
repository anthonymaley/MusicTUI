## Key Decisions

- **Skill-only surface (supersedes the morning's transport-only lane split).** Transport's determinism argument died against hardware: media keys are instant, global, zero-setup, and outside the plugin entirely. The deciding insight: the skill never needed to be deterministic — it needed deterministic TOOLS. Parsing moved to the CLI (PlayParser); the model turn is forwarding, not judgment.
- **Named speakers in `music play` = exclusive group**, matching the natural reading of "play X in the kitchen and living room". Additive single-speaker behavior retired with a release note.
- **Resolution order: whole-query first, song+artist split as fallback** — `contains`-based artist matching with short fragments is too greedy to run first.
- **No osascript fallback in the skill.** A missing CLI gets an honest "run scripts/install.sh" instead of a degraded silent path. The zero-setup tier was retired knowingly: it solved a problem media keys already solve.
